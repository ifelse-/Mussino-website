Producer Top 5 Features
A session is a contest you control. Credits are used to create sessions for songwriters to enter.

  • Create music sessions (Upload Beats)
  • Set session fee for songwriters
  • Sell beats
  • Earn percentage from session sales
  • Track earnings each month

Songwriter Top 5 Features
Songwriters can enter any amount of sessions using credits for paid royalties.

  • Enter Text, Audio, or Video session
  • Convert 16 bars lyrics into cash
  • Sell music
  • Earn percentage from session sales
  • Track royalty earnings each month

Who are the Judges?
Judges earn points that can be use towards future events and prizes.

  • Fans
  • Family
  • Website Visitors
  • Social Media Followers
  • TELL A FRIEND!

How do I get paid? A percentage is split between Producers and songwriters with the most talented votes.
Ex: 1 Producer session + 100 songwriters = $100 Royalties amount.

  • Producers earns 25% of each session
  • Songwriters earns 35% of each session
  • Selling music
  • Selling audio session
  • Selling Beats
