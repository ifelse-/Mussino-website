HOW TO INSTALL:
-----------------------------

# Unzip the file you downloaded from CodeCanyon
# Upload the "send_to_friend/" folder to your website (eg: yourwebsite.com/send_to_friend/)
# Add this script tag to your website:

<script language="javascript" src="/send_to_friend/popup.php?js" type="text/javascript"></script>

# Create a link on your website like this:

<a href="/send_to_friend/popup.php" title="Send to Friend" class="sendtofriend">Send Page To Friend</a>

# Done!

  (replace yourwebsite.com with your actual website above)
